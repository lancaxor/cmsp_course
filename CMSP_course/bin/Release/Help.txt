Справка по приложению CMSP_course
----------------------------------

Приложение предназначено для моделирования Гамма Секанс распределения методом Неймана, методом обратных Функций и аналитическим методом.
Для лучшего сравнения, лучше выполните моделирования в такой последовательности:
1) Метод Неймана
2) Метод обратных функций
3) Аналитический метод
Для сравнения достаточно использовать значения по умолчанию.
Если требуется изменить параметры, то следует иметь в виду, что изменение параметров Сигма и Мю на вкладке Метода обратных функций влияет и на остальные методы, прочие же параметры предназначены только для того метода, на вкладке которого они задаются.

Системные требования:
ОС Windows 7/Vista/Server2008/8/8.1
Компонент Microsoft Chart Controls for Microsoft .NET Framework 3.5
Для редактирования проекта понадобится Visual Studio 2008 или выше с установленным компонентом Microsoft Chart Controls Add-on for Microsoft Visual Studio 2008
ОЗУ: минимум 200МБ
Процессор: 900 МГц или выше
